Who We Are

This is a project for the Coursera Full-Stack Web Development Specialization. It is an app for clubs, churches, etc. to connect members. Users can add or update their own info, and admins can update anyone's info. Built with:

  • Angular JS
  • Node JS
  • Mongo DB
  • Bootstrap
